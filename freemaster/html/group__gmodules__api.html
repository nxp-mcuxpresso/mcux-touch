int32_t nt_module_change_mode ( struct nt_module *  module,
const enum nt_module_mode  mode,
const struct nt_electrode *  electrode 
)
Parameters
modulePointer to the module.
modeNew requested mode of the module.
electrodePointer to the electrode used in special modes (low-power & proximity); only one electrode is enabled in these modes.
Returns
  • NT_SUCCESS if the mode was properly changed
  • NT_FAILURE if the mode cannot be changed This is an example of changing the mode of the module operation in the NT library:
    if(nt_module_change_mode(&my_nt_module, NT_MODULE_MODE_PROXIMITY, &my_proximity_electrode) == NT_FAILURE)
    {
    printf("The change of mode for my_nt_module failed.");
    }
