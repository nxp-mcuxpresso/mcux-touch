Data Structure Documentation

struct nt_module_safety_interface

safety interface structure; safety uses this structure to register the entry points to its algorithms. This approach enables a kind-of polymorphism in the touch System. Safety functions are processed the same way from the System layer, regardless of the specific implementation.
