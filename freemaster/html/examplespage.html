FRDM touch-haptic demo for FRDM-KE15Z

This demo aims to provide feedback to the user's interaction by vibration or sound. This demo is intended for the FRDM-KE15Z board combined with the FRDM-HAPTIC and FRDM-TOUCH plug-in boards. The haptic demo can run in the following configurations:

  • Without the FRDM-TOUCH board (two electrodes only), requires adding the definition "#define NT_FRDM_TOUCH_SUPPORT 0" into the main.h file.
  • Without the FRDM-HAPTIC board (basic touch-sensing demo), requires adding the definition "#define NT_FRDM_HAPTIC_SUPPORT 0" into the main.h file.

Two electrodes placed directly on the FRDM-KE15Z board can trigger haptic feedback (short touch) and change haptic modes (long touch) have the following functionality:

  • The electrode E1 sets the cyan color after the touch event and changes the audio mode variant:
    • Beep - typical "beep" sound
    • On - standard "tap" sound
    • Off - no audio
  • The electrode E2 triggers haptic feedback (short touch) and changes the vibration mode (long touch):
    • On - vibration enabled
    • Off - vibration disabled

Mode changing is made by continuously holding a given button, which sets the next audio or vibration mode. The modes are changed in the loop of three audio or two vibration modes. The new mode is signalized by triggering feedback with the newly set mode.

The four arrow electrodes of the keypad are placed on the FRDM-TOUCH to trigger haptic feedback. Their electrodes use mutual sensing capacity. All these six electrodes can create touch/release/autorepeat/multi-touch events. The rotary controls trigger haptic feedback when moving the finger and its electrodes use self-capacitance measurement. The slider controls trigger haptic feedback when moving the finger and changing the haptic feedback force depending on finger position (only for the slider). The rotary and the slider can create initial touch/movement/release events. When a movement event is recognized, the variables "rotaryPosInsens" or "sliderPosInsens" from the ui_control.h file defines the minimal position change to trigger haptic feedback. The FreeMASTER GUI in the "Controls" tab monitors the slider, rotary, keypad state, position, and movement direction.

FRDM-HAPTIC board contains SW1 switch with 6 positions for configuration:

SW # Selection Description
1,2 0,0 RESERVED, no signal
0,1 RESERVED, no signal
1,0 Main Board: RT1040/50/60
1,1 Main Board: FRDM-KE15, FRDM-KE16, FRDM-KE17, RT1160/70, RT595
3 0 Motor M30 enabled
1 Motor M30 disabled
4 0 Motor M40 enabled
1 Motor M40 disabled
5 0 Motor VCC set to 3.3V
1 Motor VCC set to Main Board VDD
6 0 Motor VS set to 5V
1 Motor VS set to Main Board VIN

FRDM touch-sensing demo for FRDM-KE15Z, FRDM-KE16Z, FRDM-KE17Z or FRDM-KE17Z512

This demo is intended for the FRDM-KE15Z, FRDM-KE16Z, FRDM-KE17Z or FRDM-KE17Z512 baseboards combined with the FRDM-TOUCH plug-in board. This application can work without the FRDM-TOUCH board (two electrodes only), which requires adding the definition "#define NT_FRDM_TOUCH_SUPPORT 0" into the main.h file. The two touch electrodes (placed directly on the FRDM-KE15Z or FRDM-KE16Z boards) are used to set the LED to purple or azure colors.

There is an RGB LED controlled by controls from the FRDM-TOUCH board or the FRDM-KE15Z (FRDM-KE16Z, FRDM-KE17Z or FRDM-KE17Z512) board (four plus two-button keypad, rotary, and slider). The rotary controls the hue (color) of the LED and its electrodes use self-capacitance measurement. The slider controls the brightness of the LED and its electrodes use self-capacitance measurement. The buttons set the LED to the red/green/blue/white colors with full brightness and their electrodes use mutual capacity sensing. The FreeMASTER GUI in the "Controls" tab monitors the slider, rotary, button state, position, and movement direction.

FRDM touch-sensing demo for FRDM-K32L2A4S

This demo is intended for the FRDM-K32L2A4S board. The two electrodes create the application for freedom boards with an on-board analog slider. The on-board LED is turned on when the touch event is detected and is turned off when the ﬁnger release event is detected. The LED toggles when the ﬁnger moves on the analog slider, showing that the position is being changed. The color of the LED indicates the slider position and is changing from red (low position detected) through yellow, green, and purple to blue (high position detected) color.

FRDM or EVK touch-sensing demo for MCXN947

This demo is intended for the FRDM-MCXN947 or MCX-N9XX-EVK(MCX-N5XX-EVK) boards. The application for freedom boards with the one-touch button (one electrode only) or EVK with an on-board analog slider (two electrodes).

  • The FRDM-MCXN947 on-board LED is turned on when the touch event is detected, and it is turned off when the ﬁnger release event is detected. The electrode uses self-capacitance measurement.
    • The FRDM-TOUCH plug-in board extends the demo. The four arrow electrodes of the keypad placed on the FRDM-TOUCH set the LED to red/green/blue/white color and keep that color until the finger is released. Also, color mixing can be achieved if two arrows are pressed simultaneously. The four arrow electrodes of the keypad use mutual capacity sensing. All these five electrodes can create touch/release/autorepeat/multi-touch events.
  • The MCX-N9XX-EVK on-board LED is turned on when the touch event is detected, and it is turned off when the ﬁnger release event is detected. The LED toggles when the ﬁnger moves on the analog slider, showing that the position is being changed. The color of the LED indicates the slider position and is changing from red (low position detected) through yellow, green, and purple to blue (high position detected) color.
    • The FRDM-TOUCH plug-in board extends the demo. The four arrow electrodes of the keypad placed on the FRDM-TOUCH set the LED to the red/green/blue/white colors with full brightness and their electrodes use mutual capacity sensing. All these five electrodes can create touch/release/autorepeat/multi-touch events.

DK touch sensing demo for QN908xCDK

This demo is intended for the NXP QN908xCDK board combined with the Touch Sense board V1 plug-in board. There is an RGB LED controlled by the Touch Sense board V1 controls (four-button keypad, rotary, and slider). The rotary and slider control the brightness of the red LED. The "up" keypad button switches on the blue LED, the right or left buttons toggle the blue LED, and the down button switches off the blue LED. The Touch Sense board V1 and the example enable the switching between the touch elements (by default, only the rotary and keypad are activated). Activate the slider and circle button keypad using the following configuration:

switch elements
Rotary + Keypad: SW2: 1-OFF, 2-OFF, 3-ON, 4-ON
SW3: 1-ON, 2-ON, 3-OFF, 4-OFF
SW4: 1-ON, 2-ON, 3-OFF, 4-OFF
SW5: 1-ON, 2-ON, 3-OFF, 4-OFF
switch elements
Slider + CS Keypad: SW2: 1-ON, 2-ON, 3-OFF, 4-OFF
SW3: 1-OFF, 2-OFF, 3-ON, 4-ON
SW4: 1-OFF, 2-OFF, 3-ON, 4-ON
SW5: 1-OFF, 2-OFF, 3-ON, 4-ON

All the electrodes in the Capacitive Sense module use self-capacitance measurement. The FreeMASTER GUI application monitors the slider, rotary, button state, position, and movement direction in the "Controls" tab.
